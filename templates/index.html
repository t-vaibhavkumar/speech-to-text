Speech to Text Transcriber

Through this website, the user can transcribe any audio either by uploading an audio file or speaking directly through the microphone and getting the text transcript in real time.

The project is built in Python language. The application makes use of a Python library to convert the speech to text. It uses sqlite3 for the database for the purpose of storing the history. Flask for the back-end structure and HTML, CSS, Javascript for front-end structures.
